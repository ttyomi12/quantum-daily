## ⚙️ 它是怎么运作的

```
每天早上 9:00（北京时间）
    ↓
GitHub Actions 云端自动唤醒（不经过公司网络）
    ↓
自动搜索全球量子科技新闻（Google News RSS，14组关键词）
    ↓
DeepSeek API 自动核验、撰写中文摘要、分类
    ↓
生成 HTML 网页 → 自动推送到 Gitee
    ↓
Gitee Pages 自动更新
    ↓
✅ 你在公司打开浏览器就能看
```

**你什么都不需要做。**

## 🔧 修改配置

如需调整关注的领域、友商或信源，编辑仓库中的配置文件：

| 文件 | 用途 |
|------|------|
| `config/sources.json` | 信源网站（增删改） |
| `config/competitors.json` | 友商清单（修改 T0 级别、关注点） |
| `config/keywords.json` | 搜索关键词 |

修改后推送，下次日报生成时自动生效。
